# 🪱 Wriggle Clicker

A *Caenorhabditis elegans* idle/clicker game — Cookie Clicker, but you culture lab worms.

Click the worm to culture worms; buy petri dishes, OP50 lawns, incubators, CRISPR
stations and genome sequencers; automate with Auto-Picker bots; level buildings with
nutrient lumps; chase golden worms; and evolve for permanent genome bonuses.

**Play:** open the GitHub Pages URL for this repo.

Single self-contained `index.html` — no build step, no dependencies. Saves to
`localStorage` (versioned + export/import, so updates never wipe progress).

Game mechanics mirror Cookie Clicker's verified constants (1.15 cost growth,
25% sell refund, Frenzy ×7/77s, cube-root prestige, etc.).
